# Red Envelope Puzzle
Possibly 8 envelopes, each of which is numbered, found in various safes throughout the manor detailing a possible incident/cover-up and possible blackmail.

What's different about this puzzle, is that once a red envelope is found, a red envelope icon is permanently shown in the player's inventory (tab) menu. This is how the conclusion of 8 letters was reached, as there appears to be room for 8 envelopes on this screen.
