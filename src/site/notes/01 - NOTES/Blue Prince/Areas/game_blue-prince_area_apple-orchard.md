# Apple Orchard
Locked behind a white gate with a combination padlock that takes 4 numbers.****

> [!success] PERMANENT UNLOCK
> Start the day with +20 #steps.

- [[#Padlock]]
- [[#Gardener's Logbook]]

### Padlock

> [!note] Padlock Combination
> **1128**

The padlock combination can be found by using a magnifying glass on the photo of the Apple Orchard found in the #darkroom. 
The photo contains the code within a heart carved onto the tree that is now cut down.
