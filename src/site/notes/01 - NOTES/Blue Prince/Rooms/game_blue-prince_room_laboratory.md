### Periodic Table of Elements
#periodic-puzzle

Two period table of elements exist on different walls of the lab. One that contains the actual elements while the other contains numbers and black squares.

This can be solved two ways:
1. By overlapping the two tables, or
2. By using an magnifying glass on the table with the numbers (this reveals the elements underneath)

By lining up the letters of the elements overlapping with the numbers in ascending order, it spells out "**PUSH THREE UP AFTER NINE**"

The periodic table with the numbers on it also has an attached sticky note pointing towards the #lab-machine-puzzle .
